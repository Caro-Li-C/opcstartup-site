01
算力券
国家人工智能券

基本介绍
算力券是国家发展改革委主导推动的政策工具，深圳由市发改委牵头落地执行，属于国家级券种在地方的承接实施。其核心目的是降低企业使用智能算力的门槛，推动人工智能产业发展。

基本信息
全称
人工智能券（算力券）
主管部门
深圳市发展和改革委员会
管理服务机构
深圳市智城西部数据科技有限公司
政策层级
国家发改委统一部署，地方承接实施
适用对象
有智能算力使用需求的企业
资助标准
按国家发改委人工智能券工作有关要求执行

补贴比例和额度以国家政策及深圳市年度实施方案为准

2026年度资格审核结果已公示（5月13日-20日）

申请流程
第一步：关注申报通知

→ 登录深圳市发改委官网（fgw.sz.gov.cn），关注"通知公告"栏目

第二步：企业自主申报

→ 按申报通知要求准备材料，向管理服务机构（智城西部数据科技）提交申请

第三步：管理服务机构审核

→ 智城西部数据科技有限公司对企业提交的申请材料进行审核

第四步：专家评审

→ 组织专家对申请企业进行评审

第五步：资格公示

→ 审核结果在市发改委官网公示（5个工作日）

第六步：获得资格后使用

→ 通过审核的企业获得算力券使用资格

申请流程
第一步：在线填报

→ 登录深圳市科技业务管理系统（sticapply.sz.gov.cn）

→ 使用项目负责人账号在线填报申请书

第二步：上传材料

→ 按申请指南要求上传申请材料的电子扫描件（复印件需加盖公章）

→ 提交审核

第三步：签字盖章

→ 点击"签字盖章页打印"，将打印文件签字盖章后扫描上传

→ 提交受理（系统状态变为"待窗口受理"）

第四步：形式审查

→ 市科技创新局对申请材料进行形式审查

第五步：专项审计

→ 对通过形式审查的项目进行专项审计（需提前签署训力审计同意书）

第六步：额度发放

→ 审核通过后发放训力券额度

第七步：兑现

→ 完成算力服务后，按要求提供训练/推理基本情况、应用场景等证明材料申请兑现

申请材料清单
• 项目申请书原件

• 深圳市训力券使用场景的情况说明（加盖公章）

• 项目负责人及两名主要成员的学位/职称证书复印件+全职在深工作证明（加盖公章）

• 2025年度财务审计报告复印件（注册未满一年提供验资报告）

• 上年度完税证明复印件（非事业单位提供）

• 项目承诺书原件（法人签名+公章）

• 训力审计同意书（法人签名+公章）

三个独立环节
模型券分为三个独立环节，需依次完成：

环节一：模型服务机构入库

由模型服务机构向市工信局申请，经审核、公示后滚动确定入库名单。

入库条件：

→ 获得国家网信办生成式人工智能模型备案的单位或其关联方

→ 提供模型服务，有明确的服务内容、服务规范和收费标准

环节二：模型券申领

由需求方（企业）向市工信局申请，经审核、公示后发放。

申领条件：

→ 深圳市内实际从事生产经营活动的企业

→ 与已入库非关联方模型服务机构签订了服务合同（协议）

→ 模型相关服务在2024年12月1日至2026年8月31日期间开展

→ 模型须是已在国家网信办备案的模型

环节三：模型券兑现

原则上每季度兑现一次，由需求方向市工信局提出申请。

兑现条件：

→ 在兑现通知规定期限内申请

→ 已完成模型服务、智能体开发应用等

→ 购买人工智能模型服务总费用不低于50万元

申请流程
环节一：模型服务机构入库

1. 登录市工信局统一信息平台

2. 在线填写入库申请书

3. 上传备案材料、服务内容、收费标准等

4. 市工信局审核

5. 公示入库名单

环节二：模型券申领

1. 登录市工信局统一信息平台

2. 在线填写申领申请书

3. 上传与非关联方入库机构签订的服务合同

4. 上传项目建设方案

5. 市工信局审核、公示

6. 发放模型券

环节三：模型券兑现

1. 模型服务完成后，在兑现通知规定期限内申请

2. 登录市工信局统一信息平台

3. 上传支付凭证、发票等

4. 上传项目总结（验收）报告

5. 市工信局审核（含专项审计、现场核查）

6. 公示兑现结果

7. 补贴资金拨付至需求方

申请材料
入库材料：

→ 生成式人工智能模型在国家网信办完成备案的有关材料

→ 大模型的基本情况和服务内容、服务规范、收费标准

→ 关联方证明材料（入库申请单位非备案单位时提供）

申领材料：

→ 与非关联方已入库模型服务机构签订的服务合同（协议）复印件

→ 项目建设方案

申请流程
第一步：认定OPC资格

→ 选择目标OPC社区

→ 联系社区运营机构，提交入驻申请（营业执照、法人身份证、经营计划书等）

→ 社区运营机构审核（通常3个工作日）

→ 签订入驻协议

→ 社区统一向区主管部门提交OPC认定申请

→ 区主管部门认定OPC资格

第二步：申请智能券

→ 经认定的OPC，向所在社区运营机构提出需求

→ 社区运营机构汇总后向区主管部门申报

→ 区主管部门审核后发放智能券或直接抵扣

→ OPC使用智能券抵扣算力/模型/语料等成本

第三步：年度复核

→ 每年复核经营情况，持续享受成长扶持

• 智能券目前没有全市统一的线上申报平台，需通过各区OPC社区运营机构线下对接

• 不同区的政策力度、兑现方式差异较大

• 智能券可与市级训力券、模型券、语料券叠加享受

• 各区OPC社区入驻标准不同，部分社区对团队成员背景有较高要求（如前海要求全球知名高校/科技企业背景）


06
科技创新券
深圳市科技创新局 · 历史最悠久的券种

基本介绍
科技创新券是深圳市历史最悠久的券种之一，由市科技创新局发放，用于支持中小微企业和创客向服务机构购买各类科技服务。与上述AI专项券不同，创新券覆盖范围更广，不限于人工智能领域。

基本信息
全称
深圳市科技创新券
主管部门

• 研究开发服务：工业（产品）设计、集成电路设计、技术解决方案、中试及工程化开发、云计算等

• 技术转移服务：技术转移、成果转化等

• 检验检测认证服务：产品检验、指标测试、产品性能测试、集成电路封装测试等

• 知识产权服务：知识产权代理、知识产权检索分析等

• 创业孵化服务

申请流程
第一步：在线填报

→ 登录深圳市科技业务管理系统（sticapply.sz.gov.cn）

→ 在线填写科技创新券申请书

第二步：准备纸质材料

→ 通过系统打印申请书纸质文件

→ 法定代表人签字并加盖公章

第三步：提交材料

→ 向市科技创新局提交以下材料（一式两份，A4胶装）：

1. 申请书纸质原件（法人签字+公章）

2. 组织机构代码证、营业执照或登记证复印件

3. 法人代表身份证复印件（加盖公章）

4. 上年度完税证明复印件

5. 上年度财务审计报告复印件（注册未满一年可不提供）

第四步：审核

→ 市科技创新局对申请材料进行审核

→ 审核周期约3-4个月

第五步：发放

→ 审核通过后发放科技创新券额度

第六步：使用

→ 在入库服务机构购买科技服务时抵扣

→ 券有效期1年，逾期自动失效

第七步：兑现

→ 服务完成后，服务机构按程序向市科技创新局兑现

申请条件
• 在深圳市依法注册、具有独立法人资格的中小微企业或创客个人

• 申领年度内有与科技研发活动相关的科技服务需求

• 科技服务需求在入库服务机构提供的服务项目内

• 前次申领额度已使用完毕

• 创客个人需通过创客空间或其他商事主体申请

• 企业规模划定标准按《中小企业划型标准规定》（工信部联企业〔2011〕300号）

• 创新券不得转让、买卖、重复使用

• 建议在有效期内将额度使用完，否则影响下一轮申请

• 不限行业、不限成立时限、不限注册资金

受理机关：深圳市科技创新局
