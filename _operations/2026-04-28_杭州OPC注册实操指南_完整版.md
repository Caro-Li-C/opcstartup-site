# 杭州OPC注册实操指南

一张图照着做，今天就把执照拿到手

**📢 重要消息：**2026年4月15日起，杭州OPC新政正式实施！现在注册一人公司，**零成本、零跑腿、一张桌子就能开公司**。本文是实操指南，**照着做就能完成注册**。

## 第一步：注册前准备

Before You Start

📋 你需要准备这些材料：

1️⃣ 公司名称（准备2-3个备选）

格式：杭州 + 字号 + 行业 + 有限公司
示例：杭州XX科技有限公司、杭州XX电子商务有限公司

2️⃣ 法人身份证信息

身份证正反面照片（清晰、完整露出四角）
实名认证手机号（与浙里办绑定的一致）

3️⃣ 注册地址材料（根据实际情况选择）

方式A：园区工位注册 → 园区入驻协议或工位确认单
方式B：自有房产 → 房产证
方式C：租赁房产 → 房产证 + 租赁合同

4️⃣ 其他信息

注册资本：认缴制，5年内实缴，填写合适金额即可
经营范围：参考同行，可让系统智能推荐

**💡 小贴士：**如果你还没有注册地址，建议先联系杭州OPC创业园区（如上城区、滨江区、余杭区等），他们可以提供工位注册服务，还能享受园区配套优惠！

Login "浙里办"

## 第三步：填写公司信息

Fill in Company Info

📝 填写内容：

① 公司名称核准

→ 选择企业类型：**有限责任公司（自然人独资）**

→ 填写行政区划：杭州

→ 填写字号：你想用的名字（如"XX"）

→ 填写行业表述：科技、电子商务、文化创意等

→ 组织形式：有限公司

→ 点击"查重"确认名称可用

② 基本信息填写

→ 注册资本：填写金额（如10万元，认缴制）

→ 营业期限：一般选"长期"

→ 经营范围：点击"智能推荐"，系统会根据行业自动生成，可适当增删

→ 注册地址：选择你所在区域，填写详细地址（园区地址或工位地址）

③ 法人/股东信息

→ 姓名、身份证号码、身份证地址（自动识别）

→ 出资比例：**100%**（一人公司）

→ 联系方式：手机号

④ 董监高信息（一人公司简化设置）

→ 董事：选择"不设立董事会，只设董事一名" → 填写法人信息

→ 监事：选择"不设监事会、监事"（一人公司可不设）

→ 经理：可由法人兼任

→ 财务负责人：可填写法人本人

→ 登记联络员：可填写法人本人

## 第四步：电子签名并提交

E-Sign & Submit

✓

提交审核

确认所有信息填写无误后，点击"提交"按钮

✓

系统初审

一般1-2小时内完成初审，你会收到短信通知

✓

登录浙里办APP → 点击"签名" → 人脸识别验证 → 手写签名确认

## 第五步：领取执照和印章

Get License & Stamp

🎉 恭喜！审核通过后

✅ **营业执照**：免费邮寄到家（约1-2个工作日）

✅ **全套印章**：公章、财务章、发票章、法人章 免费刻制并邮寄

✅ **电子营业执照**：可在微信"电子营业执照"小程序下载

✅ **税务登记**：系统自动同步，短信通知电子税务局账号

## 注册后必做事项

After Registration

1️⃣ 银行开户（重要！）

携带营业执照正本、法人身份证原件、公司章程等到银行办理基本户开户。杭州部分银行支持OPC企业快速开户，建议选择支持"对公业务"的网点。

2️⃣ 税务报到（次月15日前）

登录电子税务局（账号密码会短信发送），完成实名认证、绑定银行账户、核定税种等。小规模纳税人按季度申报，一般纳税人按月申报。

3️⃣ 记账报税（每月必须）

即使没有业务也要零申报！建议委托专业代账公司（市场价200-500元/月），避免税务异常影响公司信用。

4️⃣ 年报公示（每年6月30日前）

登录国家企业信用信息公示系统（浙江），填写上一年度经营情况。逾期未报会被列入经营异常名录，影响公司信用。

## 常见问题解答

FAQ

Q：注册需要多少钱？

A：全程零成本！营业执照、公章刻制、邮寄都是政府买单。

Q：没有地址怎么办？

A：可以入驻OPC创业园区，使用工位注册，园区会提供入驻证明，无需租赁办公室。

Q：注册资本填多少合适？

A：认缴制，5年内实缴即可。建议根据实际业务需求填写，10-100万元较为常见。

Q：一个人注册和多人有什么区别？

A：一人公司是"有限责任公司（自然人独资）"，出资比例100%，决策更高效，但需注意财务与个人账户分开。

Q：注册需要多久？

A：材料齐全的情况下，半天到1个工作日完成审核，之后1-2天收到执照和印章。
