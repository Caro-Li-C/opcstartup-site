# 上海OPC工商注册实操指引
附七个已证实地址来源

工商登记 · 创业指南

重要提示

上海没有全市统一的OPC免费注册地址政策。工商登记流程是全市一套系统，但地址获取必须下沉到各区OPC社区或指定园区。以下信息均来自政府官网（gov.cn）及国家级官媒证实，未找到官方文件的热门说法已剔除。

## 一、全市统一注册流程

**全程网办，政府规费0元**

上海市市场监管局确认：一人有限公司（自然人独资）全程通过"一网通办"办理，无需到场。

| 步骤 | 操作内容 | 关键产出 |
|---|---|---|
|  | 核名：登录"一网通办"→企业开办→"我没有名称"，提交3-5个备选字号 | 名称预核准 |
|  | 填报：选择"有限责任公司（自然人独资）"，录入法人、监事信息，勾选经营范围 | 系统生成公司章程 |
|  | 地址材料：上传园区/社区出具的合规住所证明（见下文分区获取方式） | 注册地址证明 |
|  | 电子签名：法人、监事通过"随申办"APP人脸识别+在线签名 | 签名回执 |
|  | 审核领证：市场监管分局审核，电子执照与纸质执照同步发放 | 营业执照+公章 |
|  | 税务登记：领照后30日内登录"上海市电子税务局"完成税种核定 | 税务UKey |
|  | 银行开户：预约对公账户，法人携带执照、公章、身份证原件面签 | 基本户 |

费用底线
工商登记政府规费为0元。刻章费用各区执行不一，部分区免费，以各区局实际为准。

## 三、个转企直接变更路径

**适用对象：**已登记在册的个体工商户

政策依据：上海市市场监管局等6部门文件

保留原成立日期
保留统一社会信用代码
保留字号
经营场所不变

**办理地点：**经营场所所在地市场监管所
**同步办理：**税务登记、公章刻制、社保登记、公积金缴存、银行账户变更预约可一次办结
**必备材料：**经营者签署的同意转换文件、债权债务清理完毕承诺书、原营业执照正副本（缴回）、公司章程、股东决定

不可行路径
个人独资企业目前无法直接变更为有限公司。上海市市场监管局官方明确回复："目前尚无相关流程"。

## 四、材料清单与高频驳回点

必备材料

法人、监事身份证原件
正反面照片

注册地址合规证明
园区/社区出具的备案住所使用证明

3-5个公司备选名称
格式：上海+字号+行业+有限公司

经营范围
必须通过"一网通办"系统规范表述查询勾选，首项对应主营业务

注册资本
认缴制，新《公司法》要求5年内实缴到位

高频驳回红线

人员冲突
一人有限公司股东不得兼任监事

地址虚假
使用未备案虚拟地址将被列入经营异常名录

税务逾期
领照后30日内未完成税务登记，处以罚款

经营范围超限
填写许可类项目（如教育培训、医疗美容）未办前置审批直接驳回
